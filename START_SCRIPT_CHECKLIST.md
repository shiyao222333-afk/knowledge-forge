## 🎯 脚本特点

### Windows 脚本 (`run_ui.bat`)
1. **智能检查**: 自动检查 Python、依赖、端口、Qdrant
2. **自动修复**: 自动安装依赖、关闭占用端口的程序
3. **友好提示**: 彩色输出（如果终端支持）、清晰的步骤提示
4. **错误处理**: 每个步骤都有错误检查，失败时会提示并退出
5. **灵活配置**: 可以修改脚本顶部的变量（PROJECT_DIR、PYTHON_EXE、PORT）

### macOS/Linux 脚本 (`run_ui.sh`)
1. **跨平台**: 支持 macOS 和 Linux
2. **彩色输出**: 使用 ANSI 颜色代码，清晰区分信息、警告、错误
3. **自动检测**: 自动检测浏览器打开命令（xdg-open / open）
4. **灵活配置**: 可以修改脚本顶部的变量（PROJECT_DIR、PYTHON_EXE、PORT）
5. **后台启动**: Qdrant 在后台启动，不会阻塞终端
