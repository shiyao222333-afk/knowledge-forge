齿轮设计基础手册

第一章 齿轮概述

齿轮是机械传动中最常用的零件之一，用于传递运动和动力。齿轮传动具有传动比准确、传动效率高、使用寿命长等优点。

1.1 齿轮分类

按轴线位置分类：
- 平行轴齿轮传动：直齿圆柱齿轮、斜齿圆柱齿轮、人字齿轮
- 相交轴齿轮传动：直齿锥齿轮、螺旋锥齿轮
- 交错轴齿轮传动：蜗杆蜗轮、准双曲面齿轮

按齿形分类：
- 渐开线齿轮：最常用，加工方便，互换性好
- 摆线齿轮：齿面接触应力小，适用于仪表
- 圆弧齿轮：承载能力高，适用于重载场合

1.2 齿轮基本参数

模数（m）：齿轮的基本参数，决定齿轮的大小。模数越大，齿轮越大。
齿数（z）：齿轮的齿数。
压力角（α）：标准压力角为20°。
齿顶高系数（ha*）：标准值为1.0。
顶隙系数（c*）：标准值为0.25。

分度圆直径：d = m × z
齿顶圆直径：da = d + 2ha = m(z + 2)
齿根圆直径：df = d - 2hf = m(z - 2.5)

第二章 齿轮失效形式

2.1 齿面点蚀

齿面点蚀是闭式齿轮传动中最常见的失效形式。在循环接触应力作用下，
齿面产生微小裂纹，润滑油进入裂纹后受高压作用，加速裂纹扩展，最终
导致齿面金属剥落，形成麻点状凹坑。

预防措施：
1. 提高齿面硬度（表面淬火、渗碳淬火）
2. 选用高粘度润滑油
3. 降低齿面粗糙度
4. 合理选择变位系数

2.2 齿面胶合

齿面胶合多发生在高速重载的齿轮传动中。由于齿面间压力大、滑动速度高，
产生大量热量使润滑油粘度下降，导致金属直接接触发生粘焊现象。

预防措施：
1. 采用抗胶合添加剂润滑油
2. 降低齿面粗糙度
3. 选用不同材料的齿轮副（如钢-青铜配对）
4. 进行齿面硬化处理

2.3 轮齿折断

轮齿折断是最危险的失效形式，通常导致传动立即中断。分为：
- 疲劳折断：齿根弯曲应力超过疲劳极限，裂纹逐步扩展
- 过载折断：瞬间载荷超过强度极限

预防措施：
1. 增大模数以增加齿根厚度
2. 采用正变位提高齿根强度
3. 齿根圆角处理降低应力集中
4. 选择合理的材料和热处理工艺

第三章 齿轮材料选择

常用齿轮材料：
- 45钢：调质或正火，适用于一般传动
- 40Cr：调质后齿面硬度可达HRC45-55，适用于中载齿轮
- 20CrMnTi：渗碳淬火，齿面硬度HRC58-62，心部韧性好
- 38CrMoAl：氮化处理，变形小，适用于精密齿轮
- QT600-3：球墨铸铁，减振性好，适用于低速齿轮

选材原则：
1. 满足强度和硬度要求
2. 考虑加工工艺性和经济性
3. 配对齿轮硬度差HRC30-50（小齿轮硬度高于大齿轮）
