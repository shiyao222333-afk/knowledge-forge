## 三、Citrinitas 的独特优势

### 1. 中文技术文档优化
- PaddleOCR 专门优化中文识别
- PPStructureV3 识别表格和公式
- 中文语义搜索优化

### 2. 引用粒度优化
- 表格行级拆分
- 引用编号连续不跳跃
- 精确到行的引用标注

### 3. 公式渲染
- KaTeX 服务端渲染
- 支持行内公式和独行公式
- LaTeX 语法

### 4. 分面分类体系
- 4 分面 × 28 字段（UDC 主类 + 内容类型 + 时效性 + 认知状态）
- 规则引擎 + LLM 兜底双重标注
- 置信度路由（自动入库 / 待审核 / 死信队列）

### 5. 守望文件夹
- 丢文件到 watch/ 目录自动摄入
- 支持 TXT/PDF/图片/Word 等多格式
- 失败自动进入死信队列，不静默丢失

### 6. 混合检索
- 向量语义搜索 + BM25 关键词检索 + RRF 融合
- 分面过滤 + 重排序
- LLM 合成回答 + 精确引用编号

### 7. 完全本地运行
- 数据不出本地
- 支持完全离线运行
- 隐私优先

## 四、选择建议

### 选择 Citrinitas，如果你：
- ✅ 需要处理中文技术文档
- ✅ 有截图、照片等需要OCR识别
- ✅ 文档中有大量公式和表格
- ✅ 需要精确的引用标注
- ✅ 希望数据完全本地存储
- ✅ 需要分面分类 + 自动守望文件夹摄入

### 选择 RAGFlow / Dify / FastGPT，如果你：
- ✅ 需要团队协作
- ✅ 需要工作流、Agent、MCP 等高级功能
- ✅ 不介意复杂的部署

### 选择 AnythingLLM / PrivateGPT，如果你：
- ✅ 需要简单的桌面应用
- ✅ 不需要处理技术文档和公式
- ✅ 只需要基本的问答功能

## 六、总结

**Citrinitas 的定位**：一个轻量级的、专注于中文技术文档的本地知识库管理系统，已具备完整 Web UI、分面分类、混合检索和守望文件夹自动摄入能力。

**核心竞争力**：
1. 中文技术文档优化（OCR、公式、表格）
2. 分面分类体系（4 分面 × 28 字段，规则引擎 + LLM 兜底）
3. 守望文件夹（自动监控摄入，零手动操作）
4. 混合检索（向量 + BM25 + RRF 融合 + 重排序）
5. 引用粒度优化（表格行级拆分）
6. 轻量级部署（install.ps1 + run.bat，不需要 Docker）

**适合人群**：
- 工程师、研究人员、学生
- 有中文技术文档积累的人
- 关心数据隐私的人
- 希望自动批量摄入文档的人
